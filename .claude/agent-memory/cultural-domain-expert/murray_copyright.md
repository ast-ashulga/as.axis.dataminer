A.T. Murray's Iliad translation was first published 1924 (Loeb Classical Library Nos. 170–171). Murray died 1946. The 1924/1925 text is public domain (life + 70).

The 1999 Loeb revision by William F. Wyatt is a derivative work under copyright and is NOT in the pipeline.

**Why:** The pipeline is public-domain-only at the start. Conflating "Murray/Wyatt" as a citation unit — as the original placeholder prompts did — would imply using the copyrighted Wyatt revision.

**How to apply:** All Iliad citations use "Murray 1924, {book}.{line}". Wyatt's name must not appear in citation formats or system prompts as a co-translator for this pipeline. The phase-c grounding_requirement includes an explicit note about this distinction.
